# Docker PHPSchool.io

![PHPSchool.io](https://avatars1.githubusercontent.com/u/14904751?v=3&s=200)

To start you may execute these commands in the Docker image folder.

**1. Build the image.**

    docker build -t phpschool .
    
**2. Run a container.**
  
    docker run -i -t phpschool bash
